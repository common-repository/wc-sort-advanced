== Description ==

Advanced Woocommerce Sorting options.

== Features ==

Sorting Options Available
* Sort by Alphabetical
* Sort by In Stock to Out of Stock


== Installation ==

1. In your admin panel, go to Plugins -> click the 'Add New' button.
2. Click 'Upload Plugin', choose file and install.
3. Click on the 'Activate' button to use Advanced Sort for WooCommerce right away.
